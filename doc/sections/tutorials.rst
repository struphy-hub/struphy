Tutorials
=========

All notebooks are available at https://gitlab.mpcdf.mpg.de/struphy/struphy/-/blob/devel/doc/tutorials/.

The simulation data needed in the notebooks is generated by running::

   struphy test tutorials

In case you want to run only one specific notebook tutorial with number ``N``, the needed data is generated via::

   struphy test tutorials -n N

All data generated in this way is stored in the Struphy installation path (obtained from ``struphy -p``) under ``io/out/tutorial_N``.
